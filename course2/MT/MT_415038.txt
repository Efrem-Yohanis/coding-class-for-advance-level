415038	Test Case	Verify the validation results for ABSEE deal

Manual test case - Steps and Expected Results

Precondition: User is logged in (shared steps 413501) and has access to the ABS-EE Deal Home page. A deal with uploaded ABSEE files exists.

Clean, step-by-step procedure:
1. Navigate to the ABS-EE Deal Home and locate the target deal.
   - Expected: The deal row is visible and contains 'View' and 'Delete' actions.
2. Click the 'View' button for the deal.
   - Expected: The ABS-EE deal details page opens successfully.
3. Click the 'Perform Validation' button.
   - Expected: A 'Select pair for validation' popup displays with radio buttons, OK and Cancel buttons.
4. Select a radio button option.
   - Expected: The radio button is selectable and the OK button becomes enabled.
5. Click the 'OK' button.
   - Expected: The popup closes and validation starts.
6. Check the 'Execution Status' tab.
   - Expected: A message 'ABS-EE Validation completed, CompletedSuccessfully : Finished Validation' is displayed.
7. Open the 'Validation Results' tab.
   - Expected: The tab displays Filename, Created Date, Period Start, Period End, Number of Records, Number of Errors, and Number of Warnings.

Notes:
- Allow generous timeouts for validation processing (60s+).
- Validation results should match expected values based on the uploaded data.

Record (sample script):
See `course2/TC/Test_Case_415038.spec.ts` for an automated implementation.

import { test, expect } from '@playwright/test';

test('Verify the validation results for ABSEE deal', async ({ page }) => {
  // 1. Navigate to ABS-EE Deal Home (assuming login is handled separately)
  await page.goto('https://13f-qa.azurewebsites.net/deals');
  await page.getByRole('button', { name: 'ABS-EE Deal Home' }).click();

  // 2. Locate and open target deal
  const dealRow = page.getByRole('row', { name: '548678 ar 29489 09-30-2024 01' });
  await expect(dealRow).toBeVisible();
  await dealRow.getByRole('link', { name: 'View' }).click();

  // 3. Click Perform Validation
  await page.getByRole('button', { name: 'Perform Validation' }).click();

  // 4. & 5. Select validation pair and confirm
  await expect(page.getByText('Select pair for validation')).toBeVisible();
  await page.getByRole('radio').first().check();
  await page.getByRole('button', { name: 'OK' }).click();

  // 6. Check Execution Status
  await page.getByText('Execution Status').click();
  await expect(page.getByText('ABS-EE Validation completed')).toBeVisible({ timeout: 60000 });
  await expect(page.getByText('CompletedSuccessfully')).toBeVisible();

  // 7. Open Validation Results tab
  await page.locator('span').filter({ hasText: 'Validation Results' }).click();
  await expect(page.getByText('Filename:')).toBeVisible();
  await expect(page.getByText('Created Date:')).toBeVisible();
  await expect(page.getByText('Period Start:')).toBeVisible();
  await expect(page.getByText('Period End:')).toBeVisible();
  await expect(page.getByText('Number of Records:')).toBeVisible();
  await expect(page.getByText('Number of Errors:')).toBeVisible();
  await expect(page.getByText('Number of Warnings:')).toBeVisible();
});
